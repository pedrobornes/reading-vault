# Reading Vault

Reading Vault es una plataforma digital desarrollada como un gestor de lecturas personal y comunitario, concebido como el proyecto final para el ciclo de Desarrollo de Aplicaciones Web (DAW). Este proyecto nació de una colaboración enfocada en entregar un producto funcional bajo plazos de entrega muy ajustados.

## Tecnologías utilizadas

- Frontend: React, TypeScript, HTML, CSS y Bootstrap.
- Backend: Java, Spring Boot y MySQL.
- Figma

## Contexto y estado del código

> **Nota:** Esta aplicación fue construida como el cierre formativo de DAW, priorizando el aprendizaje, el trabajo en equipo y la resolución de problemas. Debido a la estricta limitación de tiempo propia de un proyecto final, la plataforma es funcional pero contiene áreas de mejora y fallos conocidos. Se expone en este perfil con el objetivo de mostrar la arquitectura, la integración entre React, Spring Boot y MySQL, y la capacidad de desarrollar software en equipo bajo presión, entendiendo que el código es una base evolutiva y no un producto final.

## Características principales

- Catálogo y gestión de lecturas organizadas por comunidades y clubes.
- Sistema de roles, permisos y administración de usuarios dentro de los grupos.
- Seguimiento de actividad y control del estado de conexión de los lectores.
